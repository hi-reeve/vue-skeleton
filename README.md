
# Vue Skeleton Loader

A simple skeleton loader for vue 3


## Installation

Install vue-skeleton with npm

npm
```bash
  npm install @zynth17/vue-skeleton
```
yarn
```bash
  yarn add @zynth17/vue-skeleton
```
pnpm
```bash
  pnpm install @zynth17/vue-skeleton
``` 


## Usage/Examples

```vue
import { VSkeleton } from '@zynth17/vue-skeleton'

<template>
    <v-skeleton />
</template>
```
